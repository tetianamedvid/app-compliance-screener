# Setting up an AI Agent
# Source: https://docs.base44.com/Building-your-app/AI-agents.md

AI agents turn simple chats into actions in a Base44 app. They can update records, trigger workflows, connect to tools, and search the web.

## Capabilities
- CRUD on app data entities
- Call backend functions
- Trigger automations
- Search the web (Google search)
- Remember context across conversations (memory feature)
- WhatsApp integration

## AI Models available
- **Automatic** (Gemini 2.5 Flash): General-purpose, ~3 credits/message
- **Gemini 3 Flash**: Stronger reasoning, ~5 credits/message
- **GPT-5**: Complex analysis, ~15 credits/message

## Configuration
- **Guidelines**: Define behavior, persona, instructions, AI model
- **Tools**: Control data access, entity permissions, backend function access
- **Memory**: Global and/or per-user memory. Agent remembers preferences across conversations
- **WhatsApp**: Direct phone messaging (free on all plans, up to 3 agents)

## Key facts for UW
- Available on all plans including Free
- 3 credits per message with default model
- Agents auto-select which tools to use
- Can be connected to WhatsApp (1 credit per WhatsApp message)
- Memory can be global (shared) or per-user (private)
- WhatsApp supports images and voice messages
