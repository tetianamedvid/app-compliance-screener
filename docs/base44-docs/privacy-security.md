# Privacy and Security
# Source: https://docs.base44.com/Community-and-support/Privacy-and-security.md

## Compliance
- SOC 2 Type II compliant (report available under NDA)
- ISO 27001 certified
- GDPR compliant (DPA available)

## Data storage
- All servers located in the **United States**
- No option to store data in other regions
- Vendors bound by strict data processing agreements
- Transfer mechanisms: adequacy, SCCs, or DPF

## Security
- Industry-standard encryption for data in transit and at rest
- Built-in authentication with strong security protocols
- Data tables and private apps encrypted
- Data is NOT end-to-end encrypted (admins can access)
- Backend code fully secure and inaccessible from outside

## Account ownership
- App ownership belongs to the person who created it
- Base44 does not participate in ownership disputes

## Email domains
- App emails: `base44-apps.com` domain
- Platform emails: `base44.com` domain

## Abuse reporting
- Report abuse/misuse to: abuse@base44.com
- Misuse policy: https://base44.com/misuse
- Responsible use policy: https://base44.com/responsible-use

## Data controls
- Account deletion available
- App deletion: Dashboard > Settings > App Settings > Danger Zone
- Personal data requests: compliance@base44.com
