# AI Service Providers
# Source: https://docs.base44.com/Community-and-support/ai-service-providers.md

Base44 uses the following third-party AI service providers:

| Provider  | Terms of Service | Usage Policy |
|-----------|-----------------|--------------|
| Anthropic | https://www.anthropic.com/legal/consumer-terms | https://www.anthropic.com/legal/aup |
| Google    | https://policies.google.com/terms/generative-ai | https://policies.google.com/terms/generative-ai/use-policy |
| OpenAI    | https://openai.com/policies | https://openai.com/policies/usage-policies |

## Key for UW
- Apps built on Base44 use AI from Anthropic, Google, and OpenAI
- Each provider has their own usage/acceptable use policies
- Content generated through these providers is also subject to their policies
- Default AI agent model is Gemini 2.5 Flash (Google)
- Available models include GPT-5 (OpenAI) and Claude (Anthropic)
